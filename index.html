What is a Function?

Functions are the main "building blocks" of the program. They allow the code to be called many times without repetition.

📦 Reusable Code

A function is a reusable block of code that performs a specific task when called. Write once, use many times!

🔧 Building Blocks

Functions serve as the main building blocks of programs, helping organize and structure your code efficiently.

⚡ No Repetition

Instead of writing the same code multiple times, create a function and call it whenever needed.

Function Declaration Basics

🔍 Understanding the Syntax

  • function - The keyword that tells JavaScript we're creating a function
  • greet - The function name (can be any valid identifier)
  • () - Parentheses hold parameters (empty if no parameters needed)
  • {} - Curly braces contain the code that runs when function is called
  • greet() - This is how we "call" or "invoke" the function to run it

💡 Remember: Declaring a function doesn't run it - you must call it!

Parameters & Arguments

🎯 Key Concepts Explained

Parameters: The placeholders in the function definition (like variables waiting for values)
Arguments: The actual values you pass when calling the function
Default Parameters: Values used when no argument is provided (age = 25)
Template Literals: Using backticks (`) and ${} for string interpolation

Parameters are placeholders, Arguments are the actual values passed in

Return Values

↩️ How Functions Give Back Data

Functions can send data back to whoever called them using the return keyword:

  • With return: Function gives back a specific value you can use
  • Without return: Function automatically returns undefined
  • Early return: You can return from anywhere in the function
  • Multiple returns: Different paths can return different values

💡 Think of return like handing something back after completing a task!

The return keyword sends back a value. Without return, functions give back undefined.
